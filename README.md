# Task Tracker

## Description
A simple **Task Tracker web application** built with **React**.  
This app allows users to add, complete, delete, and filter tasks. It also displays the total number of tasks and the number of completed tasks.  

The app is designed to be **clean, simple, and user-friendly**.

---

## Features
- Add tasks  
- Mark tasks as completed  
- Delete tasks  
- Filter tasks: **All / Active / Completed**  
- Footer shows **total tasks** and **completed tasks**  
- Displays a **"No tasks yet!" message** when the list is empty  

---

## Technologies Used
- React  
- JavaScript  
- HTML / CSS  
- Vite (for development server)
